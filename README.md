<h2>Installation</h2>

Hashlib
```bash
pip install hashlib
```

Requests
```bash
pip install requests
```

BeautifulSoup4
```bash
$ sudo apt-get install python-bs4
``` (Python 2)

Python 3
```bash
$ sudo apt-get install python3-bs4
```


lxml
```bash
$ sudo apt-get install python3-lxml
```

```bash
$ pip install lxml
```
